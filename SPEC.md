# AI Interview Simulator

AI面试模拟器 - 大学生求职训练系统

## Features
- 多种面试场景：校园招聘、教资面试、考研复试、公务员结构化、英语口语
- 语音识别输入
- STAR法则追问
- 五维能力诊断报告
- 针对性的改进建议

## Usage
直接在浏览器中打开 index.html 即可使用。

## File Structure
- index.html - 主页面
- css/style.css - 全局样式
- css/app.css - 页面样式
- js/config.js - 配置和题库
- js/speech.js - 语音模块
- js/star-analyzer.js - STAR分析器
- js/scorer.js - 评分器
- js/app.js - 主应用逻辑
